To install the datapack, download the zip file from the link below and extract it into the 'datapacks' folder inside your world folder. If the 'datapacks' directory doesn't exist, create it.

To test that it worked, enter your minecraft world with cheats enabled and type '/datapack list'. This should show the list of currently installed datapacks.
